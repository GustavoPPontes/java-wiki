# Git Commands - Administration and Maintenance

Commands for repository maintenance and optimization.

**Cheat Sheet**:

| Command     | Description                                  |
| ----------- | -------------------------------------------- |
| git clean   | Remove untracked files                       |
| git gc      | Garbage collection / repository optimization |
| git fsck    | Verify repository integrity                  |
| git archive | Create a zip/tar archive of the repository   |
