# Git Commands - History Rewriting

Commands to modify previous commits (use with caution!).

**Cheat Sheet**:

| Command            | Description                                |
| ------------------ | ------------------------------------------ |
| git commit --amend | Amend the last commit                      |
| git rebase -i      | Interactive rebase (edit, reorder, squash) |
| git filter-branch  | Mass-rewrite history                       |
| git revert         | Creates a new commit that undoes another   |
