# Git Commands - Branching and Merging

Commands for parallel development.

**Cheat Sheet**:

| Command         | Description                                      |
| --------------- | ------------------------------------------------ |
| git branch      | List, create, or delete branches                 |
| git checkout    | Switch branches or restore files                 |
| git switch      | Modern branch switching                          |
| git merge       | Merge a branch into another                      |
| git rebase      | Reapply commits onto another base                |
| git cherry-pick | (It) applies a specific commit to another branch |
| git tag         | Create tags at specific points                   |
