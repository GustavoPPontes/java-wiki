# Git Commands - Snapshot

Basic workflow commands: modify → stage → commit.

**Cheat Sheet**:

| Command    | Description                             |
| ---------- | --------------------------------------- |
| git status | Shows the current state of the files    |
| git add    | Add files to the staging area.          |
| git commit | Commit changes to the repository        |
| git reset  | Undo changes / remove from staging area |
| git rm     | Remove files from the repository        |
| git mv     | Move or rename files                    |
| git stash  | Save changes temporarily                |
