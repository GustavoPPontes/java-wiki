# Git Commands - Collaboration and Remote Repositories

Commands for collaborating and working with remote repositories.

**Cheat Sheet**:

| Command    | Description                                |
| ---------- | ------------------------------------------ |
| git remote | Manages remote repositories                |
| git fetch  | Download remote changes without merging    |
| git pull   | First fetch, then merge the remote changes |
| git push   | Pushes commits to the remote repository    |
