# Git Commands - Inspection and History

Commands to view and investigate the project history.

**Cheat Sheet**:

| Command      | Description                                     |
| ------------ | ----------------------------------------------- |
| git log      | Shows the commit history for the current branch |
| git diff     | Shows differences between versions              |
| git show     | Displays details of a specific commit           |
| git blame    | It shows who modified each line of a file       |
| git shortlog | Summarize log by author                         |
| git reflog   | Show reflog (recovery)                          |
| git bisect   | Binary search to find bug commit                |
